## üéØ Project Overview

### What This Project Is

An **Interactive Dashboard** for McNeese State University's Department of Engineering & Computer Science, designed to run on:
- Windows computers
- Android tablets
- iOS devices
- Smart TVs (via Android/iOS)

### Main Features

1. **Home Dashboard**
   - Image slideshow
   - Upcoming events
   - Announcements
   - Quick access cards

2. **Weather Display**
   - Current temperature
   - Weather conditions
   - Updates every 10 minutes

3. **Interactive Maps**
   - Campus map
   - Department floor plan
   - Room details

4. **Student Clubs**
   - Club directory
   - Club details
   - Meeting schedules

5. **Faculty Directory**
   - Faculty listings
   - Contact information
   - Office locations

6. **AI Advisor**
   - Academic guidance
   - Degree planning
   - Course recommendations

7. **Academic Catalogue**
   - Program information
   - Course listings
   - Degree requirements
